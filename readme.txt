Allows you to control and add a Facebook Like Box as a widget

== Description ==

Allows you to have complete control of the Facebook Like Box iframe that gets inserted into your code.  You can control all of your options from the admin area.


== Installation ==

This section describes how to install the plugin and get it working.

1. Upload directory `Control Facebook Like Box Widget by Beyond 5280` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the widget in the widget area

== Frequently Asked Questions ==

= Is there an option to use the XFBML instead of the iframe =

No.  Only the iframe option is currently supported.
